# Emindera Docs

Official Emindera documentation, quickstarts, API reference, and integration guides.

Emindera is a React and React Native plugin. It makes messages, comments, notes, and posts feel instant by adding optimistic client state and a durable mutation queue on top of an existing GraphQL API or database-backed server.

## Status

Early alpha. These docs track the SDK in `emindera/emindera` and may change before a stable `1.0`.

## Development

```sh
pnpm install
pnpm dev
pnpm build
```

The site uses Astro Starlight with a clean, Expo-docs-inspired feel: restrained sidebar, clear prose, fast search, and focused examples.

## Related Repos

- `emindera/emindera`: Plugin packages for React, React Native, GraphQL, storage, devtools, and tests.
- `emindera/examples`: One comparison demo showing plain GraphQL/database flow versus Emindera-enhanced flow.

## What Emindera Is Not

Emindera is not an Instagram plugin, Meta automation tool, scraper, bot, private API wrapper, account automation library, or fake engagement system.
