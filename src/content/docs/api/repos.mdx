There are only three active local repos now:

| Repo | Why it exists |
| --- | --- |
| `emindera` | The plugin packages. This is what gets published to npm. |
| `docs` | The public docs site. |
| `examples` | One comparison demo that shows the value clearly. |

Removed from the local shape:

- `create-emindera-app`: removed because scaffolding made Emindera feel like a framework.
- `benchmarks`: removed because it was premature for the current product story.
- `rfcs`: removed because it was process-heavy for a simple plugin.
