---
title: React
description: Provider and hooks for existing React apps.
---

Use `@emindera/react` in an app you already have.

Main exports:

- `EminderaProvider`
- `useEmindera`
- `useEminderaSnapshot`
- `useMutationQueue`
- `useNetworkState`
- `useEminderaThread`
- `useSendMessage`
- `useComments`
- `useCreateComment`
- `useNote`
- `useUpdateNote`

These hooks are deliberately small. They do not manage routing, pages, layouts, server rendering, or app state outside Emindera's queue/local-store loop.
